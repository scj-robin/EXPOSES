\newpage
\section{Random graphs}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\paragraph{Notation and definition.} Given a set of $n$ vertices ($i = 1..n$),
$X_{ij}$ indicates the presence/absence of a (non oriented) edge
between vertices $i$ and $j$:
$$
X_{ij} = X_{ji} = \Ibb\{i \leftrightarrow j\}, 
\qquad X_{ii} = 0.
$$
The random graph is defined by the join distribution of all the
$\{X_{ij}\}_{i, j}$.

%\vspace{1cm}
\paragraph{Typical characteristics.} 
\vspace{-0.5cm}
\begin{description}
\item[\textblue{\sl Degree:}] $K_i = $ number of edges reaching
  vertex $i$.  \vspace{-0.5cm}
\item[\textblue{\sl Clustering coefficient:}] probability for two vertices connected
  to a third one to be connected with each other ($c = \Pr\{i
  \leftrightarrow j \;|\; i \leftrightarrow k, j \leftrightarrow k\} =
  \Pr\{\nabla \;|\; \Vsf\}$).  \vspace{-0.5cm}
\item[\textblue{\sl Diameter:}] Longest path between two vertices.
\vspace{-0.5cm}
\item[\textblue{\sl Number of occurrences of some motifs:}] distribution of the
  number of occurrences of a given motif: triangles ($\nabla$),
  $\Vsf$, stars ({\Huge $\Starsf$}), etc.
\end{description}

\newpage
\section{Erdös-Rényi mixture for graph (ERMG)} \label{Sec:ErdosMixture}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\paragraph{Erdös-Rényi (ER) model: 'Random graph'.} 
\vspace{-0.5cm}
\begin{itemize}
\item All pairs of vertices  are connected with same probability
  $\pi$.
\vspace{-0.5cm}
\item The edges are independent. 
\vspace{-0.5cm}
\end{itemize}
This model fits poorly many real-world networks, probably because
of some heterogeneity between the vertices.

% %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
% \subsection{An explicit random graph model} 

\paragraph{ERMG model: Mixture model.}
\vspace{-0.5cm}
\begin{itemize}
\item The edges are spread into $Q$ groups (labels) with respective
  proportions $\alpha_q = \Pr\{i \in q\}$.
  %\vspace{-0.5cm}
\item A vertex with label $q$ and a vertex with label $\ell$ are
  connected with probability $\pi_{q\ell} = \Pr\{i \leftrightarrow j
  \;|\; i \in q, j \in \ell\}$.  \vspace{-0.5cm}
\item The edges are independent given the labels of the vertices.
\end{itemize}
A high value of $\pi_{q\ell}$ reveals a preferential connectivity
between groups $q$ and $\ell$.

% poorly many real-world networks. It actually corresponds to $Q = 1$
% group.

\newpage
\subsection{Some properties of the ERMG model}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\paragraph{Conditional distribution of the degree:} If vertex $i$
has label $q$, its degree has a Poisson distribution
$\Pcal(\lambda_q)$ where $\lambda_q = (n-1) \sum_{\ell} \alpha_{\ell}
\pi_{q\ell}$.

\bigskip\bigskip
\paragraph{Marginal distribution of the degree:} If the label of
vertex $i$ is unknown, the distribution of its degree is a mixture of
Poisson distributions:
$$
K_i \approx \sum_q \alpha_q \Pcal(\lambda_q).
$$

\paragraph{Clustering coefficient:} Its value depends on all the
parameters of the model 
$$
c = \Pr\{\nabla \;|\; \Vsf\} = \Pr\{\nabla\} / \Pr\{\Vsf\} 
= \frac{\sum_{q, \ell, m} \alpha_q \alpha_{\ell} \alpha_m \pi_{q\ell}
\pi_{qm} \pi_{\ell m}}{\sum_{q, \ell, m} \alpha_q
  \alpha_{\ell} \alpha_m \pi_{q\ell} \pi_{qm}}.
$$

\section{Estimation of the parameters}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\paragraph{Aim.} Given an observed graph we want to
\begin{itemize}
\vspace{-0.5cm}
\item estimate the proportions $\alpha_q$ and connection probabilities
  $\pi_{q\ell}$,
\vspace{-0.5cm}
\item class each vertex into a group, or at least give the probability
  for the vertex $i$ to belong to group $q$:
  $$
  \tau_{iq} = \Pr\{ i \in q \;|\; \mbox{observed connections}\}.
  $$
\end{itemize}

\paragraph{Maximum likelihood approach.} The standard strategy for
mixture models is the E-M (Expectation-Maximization) algorithm which
alternatively
\begin{itemize}
\vspace{-0.5cm}
\item Estimates the probability for vertex $i$ to belong to group $q$:
  $ \tau_{iq} = \Pr\{ i \in q \;|\; \Xcal\}$. 
  \vspace{-0.5cm}
\item \vspace{-0.5cm} Calculates the maximum likelihood estimates of
  the parameters.
\end{itemize}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\newpage
\paragraph{Case of the ERMG model.} 
\begin{itemize}
  \vspace{-0.5cm}
\item The log-likelihood of the observed data (edges) is not
  calculable: it requires to sum over all possible labels.
  \vspace{-0.5cm}
\item The 'posterior' probabilities $\tau_{iq}$ can not be calculated.
  They are approximated using a variational approach.
  \vspace{-0.5cm}
\item The parameters estimates are :
  $$
  \widehat{\alpha}_q = \sum_i {\tau}_{iq} / n, \qquad
  \widehat{\pi}_{q\ell} = \sum_i \sum_j {\tau}_{iq}{\tau}_{j\ell}
  X_{ij} \left/ \sum_i \sum_j {\tau}_{iq}{\tau}_{j\ell} \right..
  $$
\end{itemize}

selected using a ICL criterion. (inspired from BIC).

\section{Conclusions}

\paragraph{Past.} 
\begin{itemize}
\item \vspace{-0.5cm} The ERMG model is a flexible generalization of
  the ER model and a promising alternative to the scale-free model.
\item \vspace{-0.5cm} It seems to fit well several real-world networks
  %(Airport network, Enzyme network in bacteria, etc.).
\item \vspace{-0.5cm} It is properly defined, so its properties can be
  properly studied.
\end{itemize}

\paragraph{Future.}
\begin{itemize}
\item \vspace{-0.5cm} Study the probabilistic properties of the ERMG
  model (diameter, sampling properties, etc).
\item \vspace{-0.5cm} Study the statistical properties of the
  estimates; Derive a relevant criterion to select the number of
  groups.
\item \vspace{-0.5cm} Extension to valued graphs: $X_{ij}$ not only
  0/1, but some measure of the connection intensity.
\end{itemize}

\section{Approximate distribution of the count}

Based on simulations and analogy with motif statistics in sequences,
we propose to use the \paragraph{compound Poisson distribution}
(Polya-Aepply) to approximate the distribution of $N_{\mbf}$.

The parameters of this distribution are derived from $\Esp N_{\mbf}$
and $\Var N_{\mbf}$.

        0.004 & 0.037 \\
